An Extension for Google Chrome

Download serialized DOM with a click, which should include changes 
within the page effected since loading e.g. by JavaScript.

Built version ready to install from
https://chrome.google.com/webstore/detail/download-serialized-dom/bphpkjcfblopmmmmpfmgglibakngcbfc .

Derived from https://github.com/srguiwiz/download-dom-xpi .
